# Кошачий Пинтерест - Документация

## 📋 Описание проекта

Интерактивное веб-приложение для просмотра котиков с возможностью добавления в избранное. Приложение использует API [TheCatAPI](https://thecatapi.com) для получения изображений котиков с реализацией бесконечной прокрутки и адаптивного дизайна.

## 🛠 Технологический стек

- **React 19** — фреймворк для построения UI
- **TypeScript** — типизация кода
- **Vite** — быстрый сборщик модулей
- **CSS Modules** — локальная стилизация компонентов
- **LocalStorage** — хранилище избранных котиков

## ✨ Возможности

✅ Просмотр котиков с бесконечной прокруткой  
✅ Добавление/удаление в избранное  
✅ Персистентное хранилище (LocalStorage)  
✅ Адаптивный дизайн (мобильные, планшеты, десктоп)  
✅ Скелетон-загрузчик изображений  
✅ Обработка ошибок загрузки  

## 🚀 Установка и запуск

### Предварительные требования
- Node.js 18+
- npm или yarn

### Шаги установки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/yourusername/frontend-challenge.git
cd frontend-challenge
```

2. **Установите зависимости:**
```bash
npm install
```



4. **Запустите проект:**
```bash
npm run dev
```

Приложение откроется на http://localhost:5173

## 📦 Команды

npm run dev       # Сервер разработки

## 🔑 Получение API ключа

1. Перейдите на https://thecatapi.com
2. Нажмите "get your api key" → "Get Free Access"
3. Укажите email на который отправить ключ и нажмите submit.
4. Создайте файл `.env` в корне проекта:
```bash
VITE_CATS_API_KEY=ваш_ключ_здесь
```
**Преимущества:** до 100 изображений за один запрос, дополнительные поля в возвращаемом объекте

### Приложение работает без ключа, но с ограничением:
- **Только 10 изображений за раз** вместо 100
- Возвращает только id, url, width и height

## ⚠️ Важное замечание про VPN

**Для корректной работы с API TheCatAPI может потребоваться VPN**, особенно если вы находитесь в регионе с ограничениями сетевого доступа. Если приложение не загружает изображения, попробуйте:

1. Включить VPN
2. Проверить, что API ключ действителен
3. Перезагрузить страницу

## 🐛 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Изображения не загружаются | Проверьте VPN и API ключ |
| Только 10 котиков | Нужен API ключ для получения 15 за раз |
